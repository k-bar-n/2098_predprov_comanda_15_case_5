Подробное описание каждого файла и папки в проекте:



```
..\2098_predprov_comanda_15_case_5\
```
*   Это корневая папка проекта. Она содержит все остальные файлы и папки, относящиеся к проекту.

```
|---.gitignore
```
*   Этот файл указывает, какие файлы и папки Git должен игнорировать при отслеживании изменений (например, временные файлы, файлы конфигурации, личные данные).

```
|---cert.pem
|---key.pem
```
*   Эти два файла - это сертификат и ключ, необходимые для настройки HTTPS. Они обеспечивают шифрование соединения между браузером пользователя и сервером приложения.

```
|---main.py
```
*   Это основной файл Python, содержащий логику веб-приложения (back-end). Он обрабатывает запросы пользователей, управляет данными и отвечает за генерацию HTML-страниц.

```
|---README.md
```
*   Это файл, содержащий текстовое описание проекта (обычно он включает инструкции по установке, развертыванию и использованию, но пока что я его недозавершил).

```
|---soder_script.py
```
*   Скрипт на python, для автоматического создания списка файлов проекта (soder.txt) — удалю чуть позже из проекта, так как он не нужен для работы приложения, мне нужно было для разработки.

```
|---.git (служебная папка)
```
*   Эта служебная папка содержит информацию о системе контроля версий Git, а именно - историю изменений, настройки и т.д. Она позволяет отслеживать и контролировать код проекта.

```
|---.idea (служебная папка)
```
*   Эта служебная папка содержит файлы настроек интегрированной среды разработки (IDE) IntelliJ IDEA, которая используется для разработки.

```
|---.venv (служебная папка)
```
*   Эта служебная папка содержит виртуальное окружение Python. Она нужна для изоляции зависимостей проекта от других проектов и от глобальной установки Python.

```
|---data
```
*   Эта папка содержит файлы данных в формате JSON, которые использует приложение для хранения информации.

```
|   |---inventory.json
```
*   Файл, содержащий информацию о спортивном инвентаре, включая названия, количества, состояния и другие параметры.

```
|   |---inventory_assignments.json
```
*   Файл, отслеживающий, какой инвентарь и в каком количестве закреплен за каким пользователем.

```
|   |---purchase_plans.json
```
*   Файл, содержащий информацию о планах закупок инвентаря, включая количество, цену и поставщика.

```
|   |---requests.json
```
*   Файл, содержащий информацию о заявках пользователей на инвентарь, включая тип заявки (получение, ремонт, замена), статус и т.д.

```
|   |---users.json
```
*   Файл, содержащий информацию о пользователях, включая их логины, пароли и роли (администратор или обычный пользователь).

```
|   `---user_pages_settings.json
```
*   Файл, определяющий, какие страницы доступны пользователю и администратору, а также порядок их отображения.

```
|---resources
```
*  Эта папка содержит дополнительные ресурсы, такие как документы, изображения, и прочее, которые не являются частью основного кода или данных.
